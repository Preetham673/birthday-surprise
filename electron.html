Activity Sheet 11(a)

Setting Up an Electron Project

Objective:

To install and configure a basic Electron project for desktop UI development.

Tools Required:
  • Node.js
  • npm
  • Electron
  • Terminal / Command Prompt
Procedure:
  • Install Node.js and Electron using npm.
  • Run npm init in project directory.
  • Install Electron locally with npm install electron --save-dev.
  • Create index.html and main.js files.
  • Configure package.json and run with npm start.
Program (main.js): const { app, BrowserWindow } = require('electron');
function createWindow() {
  const win = new BrowserWindow({
    width: 800,
    height: 600
  });
  win.loadFile('index.html');
}
app.whenReady().then(createWindow);
