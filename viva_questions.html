Viva Questions

What is regex in JavaScript?
Regex (regular expression) is a pattern used to match character combinations in strings.
What is preventDefault()?
preventDefault() stops the default action of an event from occurring.
How do you validate password strength?
Password strength is validated using regex to check length, uppercase, lowercase, numbers, and special characters.
Can JS validation be bypassed?
Yes, JavaScript validation can be bypassed because it runs on the client side.
Why is server-side validation still required?
Server-side validation is required to ensure data integrity and security because client-side checks can be bypassed.
